Track and monitor user activity on your WordPress website. View detailed logs of user actions, including login attempts, page views, post edits, and more.

== Description ==

The **User Activity Tracker** plugin allows administrators to monitor and log user activities on their WordPress site. With this plugin, you can keep track of user behavior, improve site security, and analyze user engagement.

### Key Features:
- Track user logins and logouts.
- Monitor post and page edits.
- Record user registration activity.
- Log page and post views.
- View logs directly from the admin dashboard.
- Export activity logs to CSV for further analysis.
- Search and filter logs by user, date, or activity type.

### Use Cases:
- Enhance site security by identifying unusual user behavior.
- Monitor content updates and editorial workflows.
- Analyze user engagement on specific pages or posts.
- Keep a historical record of user activities for auditing purposes.

== Installation ==

1. Download the plugin ZIP file.
2. Upload the ZIP file via the WordPress Admin:
   - Navigate to **Plugins > Add New > Upload Plugin**.
   - Select the downloaded ZIP file and click "Install Now."
3. Activate the plugin through the **Plugins** menu in WordPress.
4. Go to the **User Activity Tracker** menu in the dashboard to view logs and settings.

== Frequently Asked Questions ==

= Does this plugin slow down my website? =
No, the plugin is optimized to log activities efficiently without impacting your site's performance.

= Can I export the logs? =
Yes, you can export the activity logs to a CSV file for further analysis.

= Is this plugin compatible with multisite? =
Currently, the plugin is designed for single-site installations. Multisite support may be added in future releases.

= What types of activities are tracked? =
The plugin tracks user logins, logouts, page views, post edits, user registrations, and more.

= How long are logs stored? =
You can configure the log retention period from the plugin's settings page.
